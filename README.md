# nfts-uinttest
#Prerequisites:
Please install or have installed the following:
nodejs and npm
Visual Studio Code

#Installation:
1-ganache-cli :
Using npm:
npm install -g ganache-cli
or, if you are using Yarn:
yarn global add ganache-cli
2- truffle : 
npm install -g truffle
3- merkltreejs :
npm install merkletreejs
4-keccak256:
npm install keccak256
